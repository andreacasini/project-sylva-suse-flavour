# Repo structure

In this repo you can find the following resources:  
**dummy-capi-unit.md:** Describes how the capi-controller is overridden in favore of rancher-turtles.  
**suse-flavored-sylva.md:** Summary of progress of the whole project as of April 2026.  
**dummy-capi folder:** Contains all necessary bits for the override to work, needs to be referenced in 'values.yaml' when deploying a management cluster.  
**suse-neuvector-init folder:** Contains all necessary bits to allow for custom version of neuvector to be identical of suse edge 3.5.  
**mgmt-values-file folder:** An example of the close-to-final configuration to create a suse-flavored sylva management cluster.
